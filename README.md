# marketing
📊 Customer Segmentation &amp; Spending Behavior Analysis 🔎 Project Overview  This project analyzes customer demographics and spending behavior using SQL, Python, and Power BI to identify meaningful customer segments and purchasing trends. The objective is to transform raw marketing data into actionable insights that support targeted marketing strategies and improved customer engagement.

The workflow includes data cleaning, feature engineering, exploratory data analysis (EDA), customer segmentation, and interactive dashboard development, enabling businesses to better understand customer characteristics and spending patterns.

🎯 Project Objectives
Analyze customer demographics such as age, income, education, and marital status
Identify customer spending behavior across different categories
Segment customers into meaningful groups using clustering techniques
Evaluate customer purchasing patterns and spending levels
Build visual dashboards for business insights and decision-making
🛠️ Tools & Technologies Used
SQL – Data querying, filtering, and aggregation
Python – Data preprocessing, analysis, and segmentation
Libraries Used:
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn (for clustering)
Power BI – Interactive dashboards and KPI visualization
